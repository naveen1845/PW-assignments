Here is the list of some HTML tags and their discription

  1. html: Defines the root of an HTML document.
  2. head: Contains metadata about the HTML document.
  3. title: Sets the title of the HTML document (displayed in the browser tab).
  4. body: Contains the content of the HTML document.
  5. h1 to h6: Define header tags, with h1 as the largest and h6 as the smallest.
  6. p: Defines a paragraph.
  7. a: Creates a hyperlink.
  8. img: Embeds an image.
